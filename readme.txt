INFORMATION RETRIEVAL - HW2

All the tasks are coded in Python, version 2.7

 ---- File "G1.txt" contains the graph required in task 1.

 ---- File "G2.txt" is the in-links file for WT2g collection.

 ---- File "G1-PageRank.txt" contains the top 50 Top pages by their docID and PageRank score of graph G1,
      sorted in terms of the final PageRank scores

 ---- File "G2-PageRank.txt" contains the top 50 Top pages by their docID and PageRank score of graph G2,
      sorted in terms of the final PageRank scores

 ---- File "G1-Perplexity.txt" contians the list of perplexity values obtained in each round until convergence for G1. 

 ---- File "G2-Perplexity.txt" contians the list of perplexity values obtained in each round until convergence for G2.
